# 【封面】

## Bitmap 點陣圖方式繪製中文標題
- 使用 Bitmap 點陣圖顯示中文標題  
- 畫面啟動後  
  → **等待使用者輸入任意鍵**  
  → 進入遊戲主流程  

---

# 【操作說明】

### 遊戲控制
- `W`：旋轉方塊  
- `A`：向左移動  
- `S`：向下加速  
- `D`：向右移動  

### 系統控制
- `ESC`：退出遊戲（會再次詢問確認）
- `Q`：直接退出程式  

---

# 【分數系統】

| 消行數 | 得分 |
|------|------|
| 1 行 | +50 |
| 2 行 | +200 |
| 3 行 | +400 |
| 4 行 | +800 |

- **Combo**：依「累積消行數」計算  
- 分數即時顯示於 **畫面左上角**
- **每 1000 分** 觸發：
  - 掉落速度加快  
  - 進入 **REAL LIFE FUNC**

---

# 【REAL LIFE FUNC】

> 考驗記憶力與臨時反應力的小遊戲  

## 題目
**哪一個顏色的面積最多？**

## 觸發條件
- 每累積 **1000 分** 進入一次

## 流程
1. 進入事件後  
   - 遊戲後方塊版面消失
2. 開始 **5 秒倒數**
3. 使用滑鼠點擊其中一個顏色作答  

## 作答規則
- 可點擊任一顏色（複數正確答案擇一即可）

## 結果影響
- 答對：只加快 **1 tick**
- 答錯：加快 **2 tick**

## 實作技巧
- 滑鼠座標即時追蹤
- **只有座標改變才重畫游標**（降低閃爍）
- 使用條件式重繪滑鼠  
  → 非整畫面刷新，提升效能
